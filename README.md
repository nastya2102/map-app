# Mapbox

React + Mapbox example app

Demo: https://nastya2102.github.io/mapbox/

## Stack Technologies

* React
* Redux
* MapBox
* Material-ui

## First start

1. Install packages using commands `npm i`
2. Run `npm start` to start app
3. Open http://localhost:3000 to view app in the browser
